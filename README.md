## Описание проекта
Данное приложение позволяет аутентифицированному менеджеру создавать команды разработчиков для деления на группы. В качестве дополнительных функций для менеджера: мониторинг задач, контролирование выполненных работ.  
Если аутентифицрованный пользователь является разработчиков/администратором в проекте, то ему будет доступна только та доска, к какой команде он причислен. Может принимать задачи для их выполнения, а также вести комментарии, закрывать или возвращать задачи в доску.
Используемые технологии: СУБД MySQL
